# BrightChamps Demo Class

## 1-Page Cheat Sheet (Quick Revision)

### Before Class
- Join 5–7 mins early  
- Use BrightChamps email + virtual background  
- Minimum class time: 40 mins  
- Ideal end: 45–50 mins  

#### If Student Absent
- Wait 5 mins → mark “student not there”  
- Wait till 40 mins → end class  

---

### 0–8 mins: Parent Interaction
- Introduce yourself  
- Ensure parent is present  
- Ask:
  - How did you find BrightChamps?
  - Expectations?
  - Coding background?  
- Give company intro  
- Set agenda  
- Ask if parent will stay  

---

### 10–15 mins: Student Engagement

#### Rapport Building
- Name, age, class, hobbies  

#### Ask Expectations
- What do you think we’ll learn today?  

#### Fun Activity
- Simon Says / Clap Jump  

#### Concept Teaching
- Coding = Instructions to computer  
- Scratch = games & animations  

#### Platform Basics
- Block Palette → code blocks  
- Script Area → coding space  
- Stage → output  
- Sprites → characters  
- Backdrop → background  

#### Core Concept
- Algorithm = step-by-step solution  

---

### 20 mins: Project
- Build project  
- Explain each step  

---

### 5 mins: Quiz + Wrap
- Did you enjoy?  
- What did you learn?  
- Appreciate student  
- Call parents  

---

### 6–7 mins: Parent Closure
- Show project  
- Give feedback  
- Ask for counselor connection  

---

### Final Step
- Connect Academic Counselor  
- Explain importance of coding  
- Smooth handover  

---
